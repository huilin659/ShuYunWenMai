## 📖 项目简介

**【项目名称】** 是一个面向青少年与社会公众的**传统文化数学互动教育平台**。我们将中华优秀传统文化（诗词、建筑、工艺、历法、音律、算学）与数学思维深度融合，通过**知识图谱 + 趣味闯关 + 互动游戏**的形式，让学习者在沉浸式体验中感受“数学之美，文化之韵”。

> 🎯 **项目定位**：新文科 × 新理科 × 数字人文 · 轻量化互动教育平台
> 🏆 **当前状态**：已推荐参加【省赛/国赛】评选

## ✨ 核心功能

| 模块 | 功能说明 |
|------|----------|
| 📚 **六大知识库** | 诗词数学、建筑几何、工艺之美、历法天文、音律数理、经典算学，共30+知识卡片，含图文、视频、图片轮播 |
| 🧩 **趣闯关** | 六大板块共30道选择题，即时判分（正确✅绿色 / 错误❌红色），每题10分，满分300分 |
| 🎮 **互动游戏** | 节气轮盘拖拽、诗词对仗匹配、营造比例尺计算、黄金比例探索、算筹开方术（配方法可视化） |
| 🛠️ **实践工坊** | 七巧板诗词拼图（3关）、榫卯结构探索、对称剪纸等9个动手实践项目 |
| 📱 **响应式设计** | 适配PC、平板、手机，触屏拖拽操作流畅 |
